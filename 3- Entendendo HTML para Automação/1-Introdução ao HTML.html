----------------------------- HTML, ou HyperText Markup Language, é a linguagem padrão usada para criar e estruturar páginas da web. Ela fornece a base para a construção de documentos na web, permitindo que você organize texto, imagens, links e outros conteúdos. ### Estrutura Básica do HTML Um documento HTML típico tem a seguinte estrutura: ```html Título da Página

Bem-vindo ao HTML!

Esta é uma introdução ao HTML.

Clique aqui para visitar um exemplo. ``` ### Componentes Principais 1. **``**: Declara que o documento é um HTML5. 2. **``**: A raiz do documento HTML. 3. **``**: Contém metadados sobre o documento, como o título e links para folhas de estilo. 4. **``**: Define o título da página, que aparece na aba do navegador. 5. **`<body>`**: Contém todo o conteúdo visível da página, como texto, imagens e links. ### Elementos Comuns - **Cabeçalhos**: `<h1>`, `<h2>`, `<h3>`, etc., usados para títulos e subtítulos. - **Parágrafos**: `<p>` para texto em parágrafos. - **Links**: `<a href="URL">Texto do Link</a>` para criar hyperlinks. - **Imagens**: `<img src="URL da Imagem" alt="Descrição da Imagem">` para inserir imagens. - **Listas**: `<ul>` para listas não ordenadas e `<ol>` para listas ordenadas. ### Conclusão HTML é a base da web e aprender seus fundamentos é essencial para qualquer pessoa interessada em desenvolvimento web. Com HTML, você pode estruturar e apresentar informações de maneira organizada e acessível.
